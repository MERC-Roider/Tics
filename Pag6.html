¿Qué es la Programación?

La programación es el proceso de diseñar y escribir instrucciones (código) que una computadora puede ejecutar para resolver problemas o realizar tareas específicas. Estas instrucciones se escriben en lenguajes de programación, que son lenguajes formales diseñados para comunicarse con las máquinas de forma precisa. En la capacitacion de tics aprenderas el manejo de compliladores como lo es c++ en el embarcadero dedicado, realizando y ejecutando programas basicos, como una calculadora.

Conceptos Fundamentales

Algoritmo
Secuencia ordenada y finita de pasos que resuelven un problema específico.
Variable
Espacio en memoria que almacena un valor que puede cambiar durante la ejecución.
Condicional
Estructura que permite ejecutar código según si una condición es verdadera o falsa (if/else).
Ciclo o Bucle
Estructura que repite un bloque de código mientras se cumpla una condición (for, while).
Función
Bloque de código reutilizable que realiza una tarea específica.

Paradigmas de Programación

  • Programación estructurada: Uso de secuencias, condicionales y ciclos.
  • Programación orientada a objetos (POO): Organización del código en clases y objetos.
  • Programación funcional: Uso de funciones puras y sin efectos secundarios.
  • Programación web: Desarrollo de aplicaciones accesibles desde el navegador.

Pasos para Crear un Programa

  1. Definir y comprender el problema a resolver.
  2. Diseñar el algoritmo (diagrama de flujo o pseudocódigo).
  3. Escribir el código en el lenguaje elegido.
  4. Probar el programa e identificar errores (depuración).
  5. Corregir errores y optimizar el código.
  6. Documentar y publicar el programa.
